# Sales Data Analyzer

## Description

This project is a **Python-based Data Analysis and Visualization program** that analyzes sales data from a CSV file.
It uses **Pandas** for data analysis and **Matplotlib & Seaborn** for data visualization.

The program is menu-driven and allows users to explore data, perform operations, handle missing values, and generate charts.

## Features

* Load dataset from CSV file
* Explore dataset (head, tail, columns, data types)
* Perform DataFrame operations
* Handle missing data
* Generate descriptive statistics
* Create different charts for visualization
* Save charts as image files

## Charts Included

* Bar Chart – Total Sales by Department
* Line Chart – Sales Trend by Year
* Scatter Plot – Department vs Salary
* Pie Chart – Sales by Region
* Histogram – Salary Distribution

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## How to Run

1. Install required libraries
   pip install pandas matplotlib seaborn

2. Run the program
   python sales_data_analyzer.py

3. Load a CSV dataset when prompted.
